# 🎬 Mood-Based Movie Recommender
A Machine Learning web application that suggests movies based on the user's emotional state.

## 🛠️ Features
- **Mood Mapping:** Translates 5+ user moods into specific movie genres.
- **Content-Based Filtering:** Uses Cosine Similarity to find the best matches.
- **Interactive UI:** Built with Streamlit for a seamless user experience.

## 🚀 How to Run Locally
1. Clone the repo: `git clone <your-repo-link>`
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
